#  Crossy Road replica

A replica of Crossy Road powered by Godot Engine

---

## 📝 Game Presentation

This is a replica of the popular mobile game Crossy Road made in Godot Game Engine and running on Windows.
It uses MagicaVoxel plugin  to import .vox files which are voxels files used for 3d models.
All assets used in this project are from me except for the font which I found in the web.

### Technical Stack
* **Language** : C#
* **Framework/Lib** : Godot Engine
* **Tools** : Godot Engine,Visual Studio

## ⚙️ Setup

Follow these steps to run the game.

### Requirement
* **OS** : Windows 10,Windows 11
* **Softwares** : Godot mono 4.4 or more
* **Dependencies** : .NET

### Setup steps

1. **Clone repo.**
2. **Open Godot and this project by selecting the Godot project file.**
3. **Run the game.**
